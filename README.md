# webshare

Webshare is like the `SimpleHTTPServer` in Python, but with nice web ui and ability to upload files

## Usage

```
webshare [--port=NUM] [PATH]
webshare --version
webshare --help

Example:
    webshare --port 8888 /var/log/
```
